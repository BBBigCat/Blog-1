## 描述

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数

```md
示例 1:

输入: 121
输出: true
示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。
进阶:

你能不将整数转为字符串来解决这个问题吗？
```

## 代码

```js
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
  // 1. 先将数字转为字符串，再将字符串转为数组并反转
  // 2. 比较 x 与反转后的值

  let str = String(x)
  let newStr = str
    .split('')
    .reverse()
    .join('')

  if (str === newStr) {
    return true
  } else {
    return false
  }
}
```

```js
// 进阶：

// 整除除余

/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
  // 1. 将 x 除余 10，除余完后再除以 10，再除余
  // 2. 比较 x 与反转后的值
  if (x < 0 || (x % 10 === 0 && x !== 0)) {
    return false
  }
  let num = 0
  while (x > num) {
    num = num * 10 + (x % 10)
    x = parseInt(x / 10)
  }

  return x === num || x === parseInt(num / 10)
}
```

## 测试代码

```js
test('测试回文数', () => {
  expect(isPalindrome(-121)).toBe(false)
  expect(isPalindrome(0)).toBe(true)
  expect(isPalindrome(10)).toBe(false)
  expect(isPalindrome(11)).toBe(true)
})
```
